# Test Plan: P&C Carrier Market Share Index

Generated by `/plan-eng-review`

Scope: Milestone 1 only, Workers Compensation + Texas retrieval.

## Affected Interfaces

- CLI: `python -m pnc_index.cli top-carriers --state Texas --line "Workers Compensation"`
- CLI: `python -m pnc_index.cli validate`
- SQLite database: `data/pnc_market_share.sqlite`

## Critical Paths

```text
Source PDF
  -> extract page text/layout
  -> identify Workers Compensation section
  -> identify BY STATE BY GROUP block
  -> parse Texas ranked rows
  -> insert rows into SQLite
  -> query top carriers
  -> show metrics with source page
```

## Edge Cases

- Company or group names wrap across PDF lines.
- `Workers Compensation` must not match `Excess Workers Compensation`.
- Premium numbers contain commas.
- Percent values may be absent or `N/A`.
- Some jurisdictions may have fewer than 10 rows.
- State total rows must not be returned as carriers.
- Parser warnings must be persisted.

## Acceptance Criteria

- Every Milestone 1 code path above has at least one test.
- Validation command fails if the extracted data cannot support the target query.
- Each returned carrier row includes source page and raw row traceability.
